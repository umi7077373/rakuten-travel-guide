## 🎯 アフィリエイトID設定

### すでに設定済み！

あなたのアフィリエイトID `4e1292eb.3ef35378.4e1292ec.6a7f65be` は以下のファイルに設定済みです：

1. **js/main.js** - 5行目
```javascript
const AFFILIATE_ID = '4e1292eb.3ef35378.4e1292ec.6a7f65be';
```

2. **全HTMLファイル** - すべての「楽天トラベルで予約」ボタンに設定済み

- **楽天トラベルトップページ**: 一般的な検索リンク

## 📝 カスタマイズ方法

### 色を変更したい
`css/style.css` の6〜11行目で色設定を変更できます：

```css
:root {
    --primary-color: #bf0000;      /* メインカラー（楽天レッド）*/
    --secondary-color: #ff6b6b;    /* サブカラー */
    --dark-color: #2c3e50;         /* ダークカラー */
    --light-color: #ecf0f1;        /* ライトカラー */
    --text-color: #333;            /* テキストカラー */
}
```

### ホテル情報を追加したい
各HTMLファイルの`.hotel-card`セクションをコピーして編集してください。

### 新しいエリアページを追加したい
既存のエリアページ（例：`area-tokyo.html`）をコピーして内容を変更してください。

## 🔧 トラブルシューティング

### リンクが動作しない
- アフィリエイトIDが正しく設定されているか確認
- ブラウザのキャッシュをクリア

### デザインが崩れる
- CSSファイルのパスが正しいか確認
- ブラウザの互換性を確認

### スマホで表示が崩れる
- viewportメタタグが設定されているか確認
- レスポンシブデザインのCSSを確認
